A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que siginifica Linguagem de Marcação de Texto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tags associadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar listas itens ordenados
<ol> e <li> Crias listas de itens não ordenados
Imagens <img> Inserir imagens
